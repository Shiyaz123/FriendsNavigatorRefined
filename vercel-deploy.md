# Deploy to Vercel

Super fast and easy!

## Option 1: Vercel CLI
```bash
npm i -g vercel
vercel
```
Follow the prompts. Done!

## Option 2: GitHub Integration
1. Push code to GitHub
2. Go to https://vercel.com
3. Import your repository
4. Auto-deploys!

## Option 3: Drag & Drop
1. Go to https://vercel.com
2. Sign up
3. Drag your folder
4. Deploy!

Your site: `your-app.vercel.app`
